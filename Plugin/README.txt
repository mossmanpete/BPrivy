The following software is required to build the BPrivy plugin:
1. Download firebreath from git://github.com/firebreath/FireBreath
2. MS Visual C++ 2010 Express
3. Windows Driver Kit 7.1.0 (WDK) http://msdn.microsoft.com/en-us/windows/hardware/hh852361
4. CMake from www.cmake.org
5. Python 2.6 or above
6. boost libraries from www.boost.org. Download source and build and install it into BPrivy/boost.
7. After all the above is done then run the firebreath\prep2010.cmd in order to make the build directory. Proceed from there.
